# Smart Home Energy Monitoring System ⚡

## Overview
The Smart Home Energy Monitoring System is an IoT-based solution that tracks real-time electrical consumption to reduce energy waste. Using an ESP32 or Python simulation, it monitors voltage and current, processes data locally, and displays actionable usage metrics through an interactive Streamlit dashboard. It enables users to identify high-consumption appliances, lower utility bills, and optimize efficiency.

## Architecture


## Tech Stack
- **Hardware/Simulation:** ESP32 / Python Simulation
- **Data Logging:** CSV / Pandas
- **Frontend:** Streamlit
- **IoT Protocol:** MQTT (Optional)

## Features
- **Real-Time Monitoring:** Instant visualization of voltage, current, and power (Watts).
- **Historical Analysis:** Logs consumption data for long-term usage tracking.
- **Cost Estimation:** Calculates energy usage impact on utility bills.
- **Anomaly Detection:** Triggers alerts for abnormal power consumption patterns.
